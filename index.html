隐私政策 — Terties (ELSCube)

最后更新：2026-06-02
生效日期：2026-06-02
开发者：ELStudio
应用：Terties（包名：com.ELStudio.ELSCube）
核心承诺：本应用不收集、不存储、不传输您的任何个人身份信息。本应用是完全离线的单机游戏。

1. 概述

ELStudio（以下简称"我们"）开发并运营 Terties 移动应用（以下简称"本应用"）。本隐私政策旨在以最高透明度说明我们处理数据的实际方式。

2. 我们收集的信息

2.1 我们不收集的信息

本应用不会收集以下任何类别的数据：

数据类别收集状态
个人身份信息（姓名、邮箱、电话）不收集
设备标识符（IMEI、MAC、广告 ID）不收集
精确或近似地理位置不收集
联系人列表不收集
相机或麦克风数据不收集
照片或媒体文件不收集
浏览历史或应用使用数据不收集
任何形式的个人数据不收集

2.2 仅存储在您设备本地的数据

以下数据仅存储在您的设备本地，我们无法访问：

  • 游戏最高分（经典模式 / 限时模式）
  • 语言设置（中文 / 英文）
  • 音量设置（音乐 / 音效）
  • 游戏模式偏好

上述数据使用 Unity PlayerPrefs 保存。卸载本应用将永久删除以上所有数据，不可恢复。

3. 第三方服务

3.1 Unity 引擎

本应用使用 Unity 引擎构建。Unity 可能会收集匿名引擎级遥测数据（设备硬件信息、崩溃报告、帧率统计），用于改善引擎性能。这受 Unity 隐私政策 约束。

3.2 未集成的服务

本应用不集成以下任何服务：

  • 广告网络（AdMob、Unity Ads 等）
  • 分析服务（Google Analytics、Firebase 等）
  • 社交媒体 SDK
  • 云存储或账号系统
  • 推送通知服务
  • 应用内支付 SDK
  • 崩溃报告服务

4. 数据共享

我们不与任何第三方共享任何数据——因为我们自身也不收集任何数据。

5. 儿童隐私

本应用不收集任何人的个人信息，包括 13 岁以下儿童。本应用内容适合所有年龄段。由于我们不收集任何数据，不构成对儿童隐私的风险。

6. 用户权利（GDPR / CCPA）

由于我们不收集任何个人数据，您实际上不需要行使 GDPR/CCPA 下的访问、删除、更正、导出等权利——我们没有任何数据可以处理。如果您有任何疑虑，请通过下方联系方式联系我们。

7. 数据安全

尽管我们不收集用户数据，我们仍采取了合理措施保护本应用代码完整性和签名密钥。安装包经数字签名确保不被篡改。

8. 数据保留

我们不收集或存储任何用户数据。存储在您设备本地的数据在卸载应用时被永久删除。

9. 政策变更

本政策如有更新，将在本页面公布。重大变更可能通过应用更新或 Play Store 列表通知。
